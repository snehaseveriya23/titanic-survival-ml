# Titanic Survival Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning techniques.

## 📊 Dataset
- Source: Kaggle Titanic Dataset  
- Features: Age, Sex, Pclass, Fare, Embarked, etc.

## Data Preprocessing
- Handled missing values  
- Converted categorical variables into numerical form  
- Removed unnecessary columns  

## 🤖 Model Used
- Logistic Regression  

## ⚙️ Steps Involved
1. Data Collection  
2. Data Cleaning  
3. Feature Engineering  
4. Model Training  
5. Model Evaluation  

## 📈 Results
- Training Accuracy: ~80%  
- Testing Accuracy: ~78%  

## 🔍 Key Insights
- Females had higher survival rate  
- Higher class passengers survived more  
- Fare influenced survival
